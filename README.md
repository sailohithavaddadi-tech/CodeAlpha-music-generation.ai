# 🎵 Music Generation with AI

## 📌 Project Overview

This project uses Artificial Intelligence and Deep Learning concepts to generate musical note sequences and save them as MIDI files. The system preprocesses MIDI music data, prepares note sequences, and uses an LSTM-based neural network architecture for music generation.

The generated music can be exported as MIDI files and played using any MIDI-compatible software.

---

## 🚀 Features

- MIDI file preprocessing using Music21
- Note sequence extraction
- LSTM-based music generation architecture
- Automatic MIDI file generation
- Flask-based web interface
- Easy-to-use project structure
- GitHub submission ready

---

## 🛠 Technologies Used

- Python
- TensorFlow / Keras
- Music21
- NumPy
- MIDIUtil
- Flask
